The Technology

📜 Ledger Scrolls

An open-source standard for storing immutable data on the Cardano blockchain. Documents, images, audio, video — permanently preserved on-chain. A library that cannot burn.

🔗 Koios API

A decentralized, community-run REST API for querying Cardano. No API key. No single point of failure. Multiple independent operators worldwide serve chain data freely.

🧩 CIP-25 Metadata

Cardano's native NFT metadata standard. Each NFT carries hex-encoded video data in its minting transaction — readable by any Cardano tool, forever.

🔐 SHA-256 Verification

Every byte is cryptographically verifiable against an on-chain hash. Trustless data integrity — mathematically provable, no authority needed.

Create Your Own Scroll

Ledger Scrolls is fully open-source. Anyone can mint immutable data on Cardano — documents, images, audio, video. Your data, permanently on-chain, readable by anyone, owned by no one. The tools, templates, and viewers are all free on GitHub.
